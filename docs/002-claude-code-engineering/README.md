## 专栏简介

这不是一份 Claude Code 入门教程，而是一门**工程化实战课**。目标是让学习者从 Claude Code 的使用者，成长为**能够驾驭 AI 的工程指挥者**。

围绕真实工程中 Agent 协作常见的痛点，课程涵盖六大模块：

| 模块 | 解决的问题 |
|------|-----------|
| **Memory** | Agent 每次对话都「从零开始」，不理解项目背景 |
| **Sub-Agents** | 单一 Agent 角色混乱、上下文污染 |
| **Skills** | Prompt 不可复用、经验无法沉淀 |
| **Hooks** | Agent 执行过程不可控、缺乏检查点 |
| **Headless** | Agent 只能在 IDE 交互，无法进入自动化流程 |
| **Agent SDK** | 只用对话方式使用 Agent，难以嵌入现有系统 |
